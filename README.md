## 공개 범위

이 저장소에는 아래 정보만 포함합니다.

- 회사명
- 도시
- 공개 가능한 수준의 진행 단계
- 공개 가능한 수준의 다음 행동
- 역할 단위 담당자 표시
- 요약형 주의사항

이 저장소에는 아래 정보를 넣지 않습니다.

- 개인 연락처
- 비공개 담당자 정보
- 세부 영업 메모
- 세부 협상 메모
- 실제 연락 원문 및 상세 이력

## 매일 보는 순서

1. 오늘 할 일 확인
2. 이번 주 일정 확인
3. AGS 검증 파이프라인 확인
4. 우선 타깃 8개와 다음 행동 확인
5. Notion으로 옮길 TASK 확인
